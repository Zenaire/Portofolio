Reika AI adalah chatbot berbasis API yang saya kembangkan menggunakan Python, dengan kemampuan mempertahankan kepribadian unik dalam percakapan. Tidak seperti chatbot generik, Reika AI dirancang memiliki gaya bicara yang flamboyan, ekspresif, dan teatrikal, seperti seorang karakter fiksi yang hidup.

🧠 Fitur Utama:

Kepribadian Karakter Kuat:

Dibekali prompt sistem yang dirancang khusus, Reika memiliki karakter flamboyan, percaya diri, dan dramatis. Dalam setiap interaksinya, Reika merespons seolah sedang tampil di atas panggung.

Ingatan Percakapan:

Menggunakan file messages.json untuk menyimpan riwayat percakapan, memungkinkan Reika mempertahankan konteks obrolan dari satu input ke input berikutnya.

Reset Memori:

Terdapat perintah khusus (reset) untuk menghapus riwayat dan mengembalikan ke kondisi awal, cocok untuk simulasi kepribadian baru.

Logging Jawaban AI:

Semua interaksi tersimpan ke dalam file log harian (riwayat_chat.txt dan jawaban_ai.txt) untuk dokumentasi dan analisis.

Efek Ketikan Dramatis:

Output ditampilkan huruf per huruf dengan jeda pendek untuk menciptakan kesan "karakter sedang berbicara" secara langsung — mendukung nuansa teatrikal Reika.

🔧 Teknologi yang Digunakan:

  • Python 3

  • API dari OpenRouter (Model: LLaMA 3 8B Instruct)

  • Library: requests, datetime, json, os, threading, tkinter

  • File I/O untuk sistem log dan memori percakapan

🎯 Tujuan Proyek:

Proyek ini saya buat untuk mengeksplorasi pembuatan AI berbasis karakter dengan kepribadian kuat dan roleplay mendalam. Tujuannya bukan hanya untuk menjawab pertanyaan, tapi juga membangun pengalaman komunikasi emosional dan ekspresif antara manusia dan AI.

🛠️ Rencana Pengembangan ke Depan:

  • Integrasi TTS (Text-to-Speech) untuk menjadikan suara Reika hidup

  • Implementasi GUI menggunakan Tkinter untuk tampilan visual yang lebih menarik

  • Penambahan filter emosi atau reaksi otomatis terhadap perintah atau kondisi tertentu

  • Sistem login agar riwayat pengguna bisa disesuaikan per identitas
